Smart Chess - A smart revolution

The main goal of this project is to create a smart chess board that enables users to:

  • Learn to play chess
  • Play against an AI
  • Make the product affordable so that it can be easily accessible to a large audience
  • Some other final goal here probably

Our Mission

Create a simple, eleganat solution that allows users to learn chess and play against an AI to train themselves and improve their chess game.

Our Plan

Currently, we have 2 teammates. In order to achieve our goals, it would be ideal to have a solid third teammate. If you have a solid hardware background, or are passionate about software please reach out to Ye Zhou or Jason Zavala.

Our Vision

Our vision is to create an affordable solution. In order to facilitate this goal, we need to keep our hardware costs low. We plan on using a Rasberry Pi, Arduino, and an LED matrix in order to light up the chess board, and finally a simple LCD screen to select different options.

Project Abstract

In this project, we combine chess-related AI programs and physical chessboards through circuit components, so as to achieve the goal that users can play games through smart chess board and AI programs.We choose to attach a magnet to each piece. When the chess piece is moved, the computer will respond by giving instructions on where to move the opponent's piece. This project will use raspberry pi to control the chess pieces, construct an LED matrix for the chessboard, and add some physical buttons to facilitate movement. Our project is a good combination of software and hardware, so it can provide most users with the possibility of playing alone without a computer, especially for children and the elderly.

Resources needed:

  • Rasberry pi 0
  • Open Source Chess API
  • Sensor array for capturing state of the board

    Potential Challenges

    The following is a list of potential challenges, both software and hardware related that we've identified as having potential to be issues.

    • Capturing the state of the board using magnetic sensors
    • Developing AI to challenge the player while staying at a competitive level
    • Instead of developing our own API we plan on using an open source chess API to capture the state of the board and return the next best state. This allows us to save time on development, as well as processing power
    • The LED matrix will require a lot of soldering, which might lead to unexpected hardware issues.
    • Our target is affordability. Other similar solutions exist, but at a high price point. Our goal is to make this as affordable as possible in order to reach a wider audience.

    This is just a list of potential challenges, other challenges may arise and we plan to address those as they come.
